# Gordian Node Promo Site

React + Tailwind promotional website for `Gordian Node`.

## Stack

- React
- TypeScript
- Vite
- Tailwind CSS

## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Notes

- UI supports Russian and English.
- Screenshots live in `public/media/screenshots/`.
- Video embed can be enabled in `src/content.ts`.
