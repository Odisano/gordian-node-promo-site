# Editing Content

Основной текст сайта вынесен в один файл: `src/content.json`.

Кроме этого, на самом сайте теперь есть встроенный `Webmaster mode`.

Что можно редактировать без изменения React-кода:

- тексты на русском и английском
- названия и порядок пунктов в навигации
- hero-блок
- карточки разделов
- changelog summaries
- подписи к скриншотам
- статьи
- видео
- ресурсы и файлы

Что менять в первую очередь:

- `src/content.json`:
  - `ru` для русского текста
  - `en` для английского текста
- `public/media/screenshots/`:
  - если захочешь заменить текущие изображения

Что можно делать прямо на сайте:

- открыть кнопку `Webmaster`
- менять тексты
- добавлять и удалять статьи
- добавлять видео по `embed URL`
- загружать изображения и файлы
- экспортировать и импортировать весь контент как JSON

Как обновить сайт после правок:

```bash
npm run dev
```

или production-сборка:

```bash
npm run build
```

Если меняется только наполнение, JSX в `src/App.tsx` трогать обычно не нужно.

Важно:

- встроенный `Webmaster mode` хранит изменения в `localStorage` браузера
- если ты добавил файлы, изображения или статьи через сайт и хочешь перенести их в другое место, обязательно сделай `Export JSON`
- для постоянной публикации изменений их потом нужно закоммитить или импортировать в нужной среде
