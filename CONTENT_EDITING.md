# Editing Content

Основной текст сайта вынесен в один файл: `src/content.json`.

Что можно редактировать без изменения React-кода:

- тексты на русском и английском
- названия и порядок пунктов в навигации
- hero-блок
- карточки разделов
- changelog summaries
- подписи к скриншотам

Что менять в первую очередь:

- `src/content.json`:
  - `ru` для русского текста
  - `en` для английского текста
- `src/content.ts`:
  - `videoEmbedUrl`, когда появится ссылка на видео
- `public/media/screenshots/`:
  - если захочешь заменить текущие изображения

Как обновить сайт после правок:

```bash
npm run dev
```

или production-сборка:

```bash
npm run build
```

Если меняется только наполнение, JSX в `src/App.tsx` трогать обычно не нужно.
